# 🌱 Carbon Tracker

A full-stack Carbon Footprint Tracker that helps users monitor, analyze, and reduce their daily carbon emissions through an interactive dashboard and personalized insights.

##  Features

-  User Authentication (JWT)
-  User Registration & Login
-  Dashboard with Carbon Statistics
-  Track Transport Emissions
-  Track Food Emissions
-  Track Energy Consumption
-  Track Shopping Emissions
-  Track Digital Footprint
-  Set Carbon Reduction Goals
-  Real-time Dashboard Updates
-  Activity History
-  Secure REST APIs using Spring Security

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/rachitmani18/carbon-tracker.git
```

### Backend

```bash
cd backend
```

Configure PostgreSQL in:

```
src/main/resources/application.properties
```

Run the application:

```bash
./mvnw spring-boot:run
```

or

```bash
mvn spring-boot:run
```

---

### Frontend

Open

```
frontend/login.html
```

using Live Server.
